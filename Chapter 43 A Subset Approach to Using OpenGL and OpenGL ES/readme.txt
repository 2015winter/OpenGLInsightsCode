All of the code for the chapter "A Subset Approach to Using OpenGL and 
OpenGL ES" can be found at the following URL:

 https://code.launchpad.net/glmark2 

Instructions for browsing the development mainline via the web or obtaining a
local copy from the source repository via the bazaar source control system are
there.  The latest released version is also always available in the "Downloads"
section of the main project page (https://launchpad.net/glmark2).
